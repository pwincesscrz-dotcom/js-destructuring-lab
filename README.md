# JavaScript Destructuring Lab

## Overview
This lab focuses on understanding and practicing data destructuring in JavaScript (ES6). Learn how to extract values from arrays and objects using destructuring syntax, including nested destructuring, default values, renaming variables, and function parameter destructuring.

## Learning Objectives
- Use array destructuring to extract values from arrays
- Use object destructuring to extract properties from objects
- Apply nested destructuring for complex data structures
- Use default values while destructuring
- Rename variables during destructuring
- Use destructuring in function parameters
- Write cleaner and more readable JavaScript code
